# ApsaraSoftware Website (React + NestJS)

This project bootstraps an Apsara company website with:

- A React frontend for a professional multi-page marketing site
- A NestJS backend serving profile data to the frontend

## Services and Ports

- Frontend (Vite): `http://localhost:2500`
- Backend (NestJS): `http://localhost:4001`

No service uses port `3000`.

## Run Locally

Open two terminals:

### 1) Start backend

```bash
cd backend
npm run start:dev
```

### 2) Start frontend

```bash
cd frontend
npm run dev
```

Then open `http://localhost:2500`.
