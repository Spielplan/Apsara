# Apsara Website

A React single-page application for the Apsara company website, built with Vite.

## Run Locally

```bash
cd frontend
npm install
npm run dev
```

Then open `http://localhost:2500`.

## Build for Production

```bash
cd frontend
npm run build
```

Static assets are output to `frontend/dist/`.

## Lint

```bash
cd frontend
npm run lint
```
